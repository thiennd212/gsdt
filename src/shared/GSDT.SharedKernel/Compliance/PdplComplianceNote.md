# PDPL Compliance Reference

**Current law:** Law 91/2025/QH15 + Decree 356/2025/NĐ-CP (Personal Data Protection Law).

**IMPORTANT:** Decree 13/2023/NĐ-CP EXPIRED 2026-01-01.
All PDPL references in code comments, documentation, and compliance checklists MUST reference:
- Law 91/2025/QH15 (effective 2026-01-01)
- Decree 356/2025/NĐ-CP (implementing regulations)

Do NOT reference Decree 13/2023 for new implementations.
